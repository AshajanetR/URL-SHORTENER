## Clone the Repository

To get started, clone the repository using the following command:


git clone https://github.com/AshajanetR/URL-SHORTENER.git` 

## Setup the Client Application

1.  Navigate to the client application directory:
    ```
    cd URL-SHORTENER/client-app/url-shortener-app 
    ```
    
2.  Initialize the client application (if not already initialized):
    ```
    npm init -y
    ```
    
3.  Install the client dependencies:
    ```
    npm install 
    ```
    
4.  Start the client application:
    ```
    npm run dev
    ```
    

## Setup the Server Application

1.  Open a new terminal and navigate to the server application directory:
    ```
    cd URL-SHORTENER/server-app 
    ```
2.  Install the server dependencies:
      ```
    npm install
    ```
   
    
3.  Start the server application:
     ```
    npm run dev
    ```
